The Project

MERLS — Monitoring, Evaluating, Researching and Learning System

MERLS gives GRIT's team a single place to manage grants, track program impact, and report to funders — replacing what used to be scattered across spreadsheets. Built with teammates Andre and Shreya over six weeks, for GRIT (Gender Rights in Tech), a Cape Town-based nonprofit.

React Node.js Express PostgreSQL Railway OpenAI GPT-4o mini Google Analytics

Feature by Feature

A guided tour

Dashboard
On login, users land on a dashboard summarizing active grants, upcoming deadlines, and key indicators at a glance.
[ Screenshot ]
Workshop Logging
Staff log workshops and events as they happen, capturing attendance and outcomes tied back to funder requirements.
[ Screenshot ]
Gantt Chart & Timeline
A visual project timeline tracks milestones and deliverables across every active grant.
[ Screenshot ]
Indicator Tracking
Program indicators are tracked over time, giving GRIT a clear picture of impact against their goals.
[ Screenshot ]
Funding Pipeline
A dedicated view tracks prospective and active funding opportunities from first contact through award.
[ Screenshot ]
Stories of Change
A space for qualitative impact stories, which funders often want alongside hard numbers.
[ Screenshot ]
AI Insights
Powered by GPT-4o mini, MERLS generates automated daily summaries of dashboard data, saving staff time on manual reporting.
[ Screenshot ]
Analytics & Reporting
Google Analytics integration and PDF export tools let staff pull polished reports for funders in a few clicks.
[ Screenshot ]
Role-Based Permissions
Funders get scoped, limited access to relevant data, while GRIT staff retain full administrative control.
